Профессиональные навыки

Специализируюсь в области разработки программного обеспечения, backend разработки на C#/.NET (ASP.NET Core, ASP.NET MVC/WebAPI, Entity Framework, LINQ, ADO.NET, WinForms, WCF, WPF). Разрабатываю программное обеспечение с использованием стека технологий Microsoft.

В разное время писал на PHP, Python, C++, C, MATLAB.

Хорошо разбираюсь в парадигме ООП, паттернах проектирования, алгоритмах и структурах данных. Умею работать с асинхронным программированием и многопоточностью.

В качестве среды разработки использую MS Visual Studio. Также люблю использовать редактор Visual Studio Code. Пользуюсь Swagger для работы с API. Работаю с Git.

Из СУБД работаю в основном с MS SQL Server. Также имеется опыт работы с PostgreSQL и MySQL.

Выпускник механико-математического факультета МГУ им. М.В. Ломоносова (окончил с отличием в 2020 г.). Специализация: геометрия и топология. Область научных интересов: алгебраическая топология (бордизмы/кобордизмы, спектральные последовательности), интегрируемые системы (биллиарды, симплектическая геометрия).
